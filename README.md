# nanalang
> A really stupid programming language.

## Installation
Make sure you have both *Git* and *Node.js* installed.
```sh
git clone https://github.com/nanalan/g.git nanalang
cd nanalang
npm install
npm link
```

## Usage
```sh
nanalang --help
```
